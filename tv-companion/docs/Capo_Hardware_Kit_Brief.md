# Capo Hardware Kit — Product Brief
**Created:** 2026-02-24 | **Status:** Research Phase | **Owner:** Sinton.ia

---

## Two Products

### Product 1: Capo Brain (Plug-and-Play)
- Pre-configured home AI box
- Parent buys it, kid talks to it, done
- No subscriptions, no cloud tokens, no data leaving the house
- Updates OTA

### Product 2: Capo Kit (Learn-to-Build)
- Hardware + step-by-step guide
- "Build your kid's AI companion in a weekend"
- Parent and kid do it together (like LEGO but it talks back)
- Each chapter unlocks a new ability
- STEM angle: schools, homeschoolers, after-school programs

## Research Needed

### Hardware Options
- [ ] Custom board vs off-the-shelf (Pi 5, Intel NUC, Orange Pi, Radxa)
- [ ] Ship our own device vs "install on what you have" (Mac Mini, old laptop)
- [ ] Cost to manufacture custom device vs retail markup
- [ ] Battery-powered portable option?

### Connectivity (Big Problem)
- [ ] WiFi only (home use)
- [ ] Bluetooth to phone/watch (outdoor portability)
- [ ] Phone tethering / hotspot
- [ ] Starlink (rural/outdoor)
- [ ] Offline mode — what can run with NO connection?
- [ ] Watch pairing (Apple Watch, kid smartwatch)

### Local AI Performance
- [ ] Which models run on <8GB RAM? (Phi-3, TinyLlama, Gemma 2B)
- [ ] Which models run on 8-16GB? (Llama 3.1 8B, Mistral 7B)
- [ ] Voice-to-text locally (Whisper small/medium)
- [ ] Text-to-speech locally (Piper, Coqui)
- [ ] Full offline voice pipeline feasibility
- [ ] Quality comparison: local 7B vs cloud GPT-4o for kid conversations

### Form Factors
- [ ] Home hub (stationary, speaker-like)
- [ ] Portable (kid carries it, backpack clip)
- [ ] Plush integration (Capo toy with brain inside)
- [ ] Watch-only (ultra-light, cloud-dependent)

### Revenue Model
1. Kit hardware (one-time: $199 / $399 / $599)
2. Course/curriculum (one-time or subscription)
3. Personality packs ("teach Capo Spanish")
4. School/bulk licensing
5. Software-only tier (bring your own hardware)
